# Road-Lanes-Detection-
Computer Vision Project: a didactic python notebook implementing a pipeline to solve Road Lanes Detection problem given real world data. In particular, each implemented function let to the user to modify the parameters characterizing each Computer Vision method, allowing to understand their impact on an image.

## Implemented Computer Vision Methods
* Gaussian Filter;
* Canny Edge Detector;
* Hough Transform;
* Probabilistic Hough Transform.

## Project Structure
* `Notebook-DEMO/`: Contains the Notebook showing a demonstration of the pipeline for a given dataset and a local backup of the dataset. 

## DATASET
Dataset is downloaded automatically from the notebook by a link and it is composed by four real-world images. A local copy of the original dataset is already provided in `Notebook-DEMO/roads.zip` in case the link in notebook expires.

NOTE: the implemented functions works with any kind of dataset!
